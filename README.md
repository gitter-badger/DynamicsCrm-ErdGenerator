# DynamicsCrm-ErdGenerator
### Version: 1.3.5
---

Tool that generates an ERD diagram for chosen entities in CRM. The diagrams can be split into separate pages for each entity.

### Dependencies

  + Visio DLL
	+ Microsoft.Office.Interop.Visio.dll
